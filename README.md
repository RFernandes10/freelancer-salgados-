# Freelancer-Salgados 🍴

Uma página web para divulgar e gerenciar o menu de salgados e doces de alta qualidade. O projeto inclui um sistema de carrinho de compras com interface moderna e interativa.

## 🚀 Demonstração

Veja o site em funcionamento: [Freelancer-Salgados](https://cardapio-coral.vercel.app/)

---

## 📜 Descrição

Este projeto foi desenvolvido para facilitar o gerenciamento e a apresentação dos produtos do **Lilinda Salgados**, com foco em experiência do usuário e design responsivo.

### Funcionalidades
- Exibição de produtos com descrição, imagens e preços.
- Adição de itens ao carrinho.
- Cálculo de preços totais.
- Formulário para preenchimento de informações de entrega.
- Interface intuitiva e responsiva.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura básica da página.
- **CSS3** (Tailwind e Font Awesome): Estilização e design.
- **JavaScript**: Manipulação de eventos e lógica do carrinho de compras.
- **Toastify.js**: Notificações interativas.
- **Vercel**: Hospedagem do projeto.
